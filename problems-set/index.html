3.0、linux百度云打开后一直缓冲无法显示登录界面解决办法：

cd ~/.local/share/baidu/baidunetdisk rm baidunetdiskdata.db

or

rm ~/.local/share/baidu/baidunetdisk/baidunetdiskdata.db
