2.linux下查看硬盘信息:

1
2

lsblk
# list block

or

1

sudo hdparm -i/I /dev/sd*

3.查看系统安装时间(linux):

1

ls -lact --full-time /etc | tail -1 | awk '{print $6,$7}'

4.get weather:

1
2

curl wttr.in/qingdao   #获取青岛天气
curl v2d.wttr.in/qingdao

访问 wttr 可以查看更多其他有趣的命令

5.change JDK version:

1
2
3

archlinux-java help
archlinux-java status
sudo archlinux-java set java-12-jdk

6.linux unzip 乱码解决:

6.1 使用unzip命令(建议)

1

unzip -O GBK/cp936 *zip   #-O为指定解压缩的字符集编码 windows默认的字符集为GBK

6.2 使用unar命令(不建议)

1

unar sth.zip   #invalid

7.node_modules检查及更新

1
2

npm install npm-check -g
npm-check   # npm-check compare with package.json to check update or new install

update package:

1
2

npm update -S
npm update -D

8.linux百度云打开后一直缓冲无法显示登录界面解决办法：

1
2

cd ~/.local/share/baidu/baidunetdisk
rm  baidunetdiskdata.db

or

1

rm ~/.local/share/baidu/baidunetdisk/baidunetdiskdata.db

9.git push origin master报错原因可能是空文件夹未初始化：

1
2
3
4

touch readme #工程文件介绍
git add . #加入文件
git commit -m 'add' #修改部分描述
git push origin master #推到github

若报错则先pull一下

1

git pull origin master #拉取云端文件
