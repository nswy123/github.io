Task2.1

Write a general Matlab function for the symbol spaced equaliser (SSE) design according to the ZF and MMSE criteria. Your function should have four input parameters

  • channel impulse response c (i.e., the normalised vector c(n)),
  • equalizer order p,
  • delay n0, and
  • signal-to-noise ratio SN R (in dB).

Your function should have five output arguments

  • impulse response of the ZF equaliser hzf ,
  • impulse response of the MMSE equaliser hmmse,
  • overall impulse response of the ZF equaliser fzf (Hint: fzf = c * hzf ),
  • overall impulse response of the MMSE equaliser fmmse (Hint: fmmse = c * hmmse ),and
  • inter-symbol interference power affecting each signal sample in the ZF equaliser,which is denoted as ISIzf and is calculated as ISIzf = ∑n, n!=n0 |fzfn |^2.

Hints:

  • SN R (in dB) is given in this exercise. It is the SN R before equalisers (as defined in Eqs. (2.25)–(2.27)).
  • You might need to convert the SN R value from a dB value to a decimal value when necessary inside your function.

function [h_zf, h_mmse, f_zf, f_mmse, ISI_zf]= lab_sse(c, p, n0, SNR)
    % 此函数接收4个输入，分别是信道冲激响应c，均衡器阶数p，延迟n0，信噪比SNR
    % 输出5个值，分别是h_zf, h_mmse, f_zf, f_mmse, ISI_zf，题目中写的很详细
    % 前四行代码去看Lab6的ppt，我会在下面贴出比较重要的一页
    m_channel_order = length(c);
    f = zeros(m_channel_order + p, 1);
    f(n0) = 1;
    C = convmtx(c.', p + 1);
    % h_zf公式
    h_zf = pinv(C) * f;
    % 单位矩阵
    identity_matrix = eye(p + 1, p + 1);
    %h_mmse公式
    h_mmse = (C' * C + 1 / SNR * identity_matrix) ^ (-1) * C' * f;
    %f_zf和f_mmse公式
    f_zf = conv(c, h_zf);
    f_mmse = conv(c, h_mmse);
    %ISI_zf公式，n不等于n0时求和
    sum = 0;
    for index = 1 : length(f_zf)
        if index ~= n0
            sum = sum + f_zf(index) ^ 2;
        end
    end
    ISI_zf = sum;
end

f_zf_local = 1;
% 初始化fmmse的位置变量

% 假设fzf的最大值为第一个值
h_zf_max = f_zf(1);
% 假设fmmse的最大值为第一个值
f_mmse_max = f_mmse(1);
% 对fzf循环判断
for index = 1 : length(f_zf)
    % 若fzf中的值大于“最大值”，则把其赋给“最大值”
    if f_zf(index) > h_zf_max
        h_zf_max = f_zf(index);
        % 记录下标
        f_zf_local = index;
    end
    % 得出fzf最大值和其下标
    f_zf_finalMax = [h_zf_max ,f_zf_local];
end
% 对fmmse进行判断，与fzf方法一致
for i = 1 : length(f_mmse)
    if f_mmse(i) > f_mmse_max
        f_mmse_max = f_mmse(i);
        f_mmse_local = i;
    end
    f_mmse_finalMax = [f_mmse_max ,f_mmse_local];
end
% 打印出fzf和fmmse的最大值及其下标
