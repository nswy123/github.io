1、Change cursor

open .spacemacs file:

1

sudo nano ~/.spacemacs

use “ctrl+w” open search，input “user-config” to find the position，and add:

1

(setq evil-emacs-state-cursor '("red" bar))

Look like:

restart spacemacs。

2、Add themes

install a package include many themes

open .spacemacs and find that:

1

(defun dotspacemacs/layers ()

Add a package

1

themes-megapack

looks like(in the end):

then you can use “alt+x”, input “load-theme”, choose a theme, I think “ujelly” is best for me, because it can protect my eyes.
