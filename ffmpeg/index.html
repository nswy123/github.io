Preface

以下所有操作需要安装ffmpeg和imagemagick软件，在manjaro或者arch中可以使用:

1

sudo pacman -S ffmpeg imagemagick

1. ffmpeg从视频中提取音频：

1
2
3
4

ffmpeg -i input.mp4 -f mp3 -vn output.mp3
# -i: input file
# -f: format
# -vn: video not

2. ffmpeg截取音频：

如果我想把aa.mp3中的1分12秒到1分42秒的地方切出来，然后存成bb.mp3，指令如下：

1
2

ffmpeg -i aa.mp3 -ss 00:01:12 -t 00:00:30 -acodec copy bb.mp3
# 从1分12秒开始一直到30秒后的位置。

3. ffmpeg转换格式flac到mp3

1

ffmpeg -i input.flac -b:a 192k -acodec mp3 -ar 44100 -ac 2 output.mp3

4. ffmpeg把多张图片制作成视频

1
2

ffmpeg -r:v 1/5 -i "%05d.png" -r:v 30 -codec:v libx264  -preset veryslow -pix_fmt yuv420p -crf 28 -an "filename.mp4"
# 制作一个每张图片显示5s，视频侦数为30FPS的视频

5. ffmpeg从视频中截取gif

1
2
3
4
5
6

ffmpeg -ss 00:00:20 -i input.mp4 -to 10 -r 10 -vf scale=200:-1 output.gif
# -ss:截取起点
# -i:输入的视频文件
# -to: 截取终点
# -r:帧速率，值越大GIF文件质量越高
# -vf: GIF的缩放大小

6. 合并GIF文件

1
2
3

convert -delay 120 -loop 0 input.gif output.gif
# -delay:GIF播放速度
# -loop 0:无限循环

7. 从JPG图片创建GIF动画

1
2
3

convert -delay 120 -loop 0 input.jpg output.gif
# -delay:GIF播放速度
# -loop 0:无限循环

8. 查看视频的编码格式

1
2
3
4
5

ffprobe -v error -select_streams v:0 -show_entries stream=codec_name -of default=noprint_wrappers=1:nokey=1 source.mp4
# 常见的编码格式有:

# h.265(HEVC)
# h.264(AVC)

9. 视频编码转换

1
2
3
4

ffmpeg -i source.mp4 -c:v h264 -c:a copy output.mp4
#将源文件source.mp4转换为h264编码的output.mp4文件
# -c:v   -copy:video
# -c:a   -copy:audio
